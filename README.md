# 最全中华古诗词数据库

## 资源介绍

本仓库提供了一个名为“最全中华古诗词数据库”的资源文件，该数据库包含了唐宋两朝的古诗词作品，涵盖了近一万四千名诗人和词人的作品。具体内容如下：

- **唐诗**：收录了近5.5万首唐诗，涵盖了唐代众多著名诗人的作品。
- **宋诗**：收录了26万首宋诗，展示了宋代诗人的丰富创作。
- **宋词**：收录了两宋时期1564位词人的21050首词作，全面展现了宋代词坛的风貌。
